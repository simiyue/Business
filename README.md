# Business Jobs Portal

A modern, responsive job listing website showcasing companies and their opportunities across Africa.

## Features

- 🎨 Beautiful card-based UI with hover effects
- 📱 Fully responsive design (mobile, tablet, desktop)
- ✨ Smooth animations and transitions
- 🎯 Clean, organized company profiles
- 🌍 Multiple job listings with location information

## Pages

- **Jobs Page** (`#jobs`) - View all available job opportunities with company details, products, and services

## File Structure

```
├── index.html      # Main HTML structure
├── styles.css      # Complete styling and responsive design
├── script.js       # Smooth scrolling and animations
└── README.md       # Documentation
```

## How to Use

1. Open `index.html` in your browser
2. Browse company listings
3. Click on tags to see services and products
4. Visit company websites via the CTA buttons
